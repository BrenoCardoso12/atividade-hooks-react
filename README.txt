COMO RODAR

1. Abra a pasta react-hooks-moderno no VS Code.
2. No terminal, execute:
   npm install
3. Depois execute:
   npm run dev
4. Abra o link http://localhost:5173/

HOOKS UTILIZADOS

- useState: controla nome, tema e contador.
- useEffect: altera o título da aba do navegador.
- useContext: compartilha o tema entre componentes.
- useRef: foca o input pelo botão.
- useMemo: hook extra, calcula a mensagem principal com base no nome.
